Grocery item accessor

Summary

The Grocery Iist Accessor does all the data operations for grocery items in the application. I wrote the majority of the code for the current versions (though they are partially based off of earlier accessors written by team member Natalie Ruckman).

Functions

Below is a list of the functions in this accessor. They all perform basic CRUD (Create, Retrieve, Update, and Destroy) operations on the GrceryItem table in the database.

  1. Get: This function retrieves a single grocery item record from the database given its unique id.
  2. Add: This function inserts a given instance of the GroceryItem class into the database.
  3. Edit: This function, given the id of an existing grocery item and an edited instance of the GroceryItem class, applies all the properties of the edited instance to the existing record.
  4. Delete: This function deletes a single grocery item record from the database given its unique id.

Finally, the highlight of this file, the GetOnlyItemsForList function

The GetOnlyItemsForList function retrieves only the grocery items associated with a specific list (see more on that topic here). This is important, as when you are looking at a specific list's contents, you should only see the items on that list. This function does that by retrieving all the grocery items with a foreign key matching the given list id.
